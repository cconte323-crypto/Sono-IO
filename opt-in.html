SMS Messaging Policy

Sono IO — Opt-In & Consent

This page explains how messages are sent through Sono IO, how recipients can stop receiving them, and how sender consent works.

How messages are sent

Sono IO is a paid service that allows a sender to send a single text message to a phone number of their choosing. The sender pays for a pack of 5 messages and provides the recipient's phone number and message content directly. Sono IO does not send marketing, promotional, or bulk messages — each message is a single, individually composed text initiated by a paying sender.

Sender consent

By using Sono IO, the sender agrees to our Terms of Service, which require that messages comply with all applicable laws and are not used for harassment, threats, spam, or illegal purposes. Sender IP addresses are logged (in hashed form) for abuse prevention.

Recipient opt-out

Every message sent through Sono IO includes opt-out instructions. A recipient can reply STOP to immediately stop receiving any further messages from Sono IO's number pool. This is honored automatically and immediately.

Sample message

Below is an example of the exact format every message sent through Sono IO follows:

Hey, just wanted to let you know the package arrived safely. Thanks!

— Sent via SonoIO.com
Reply STOP to opt out of future messages.

Message frequency

Recipients receive messages only when a sender chooses to send one to their number. There is no recurring or scheduled messaging — each message is a one-time send initiated by a human sender.
